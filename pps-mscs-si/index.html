Mathematical Structures in Computer Science

Special Issue

on

Probabilistic programming semantics

Submission | Important dates

Probabilistic programming languages serve a formalism for modelling statistical and probabilistic phenomena. The crucial new ingredient is the ability to construct models compositionally. This formalism allows one to disambiguate subtle probabilistic behaviour and explicate hidden assumptions. It also serves as a well-defined machine-readable format to enable automatic analysis of models and even execution. The rapidly spreading use of statistical modelling in data analysis and artificial intelligence has produced renewed interest in this well-established area. Probabilistic programming languages raise unique fundamental semantic challenges that push the state-of-the-art in programming language theory and related mathematics. The semantics community rose to the challenge, and in the last few years broke new ground in the semantic foundations of probabilistic programming.

We invite the community to submit articles on all aspects pertaining to probabilistic programming semantics. These encompass, but are not limited to:

  • new development in probabilistic programming semantics
  • surveys of open challenges in the area
  • compilation of an extensive existing line of work
  • mathematical and logical structures underlying probabilistic programming semantics
  • new connections between semantic structures of probabilistic programming languages
  • incorporating expressive programming abstractions into probabilistic programming
  • semantic reasoning about probabilistic programs
